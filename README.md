# Awesome client boilerplate (non-isomoprhic)

Hot reloading starter project with React and Express.

# Dependencies
- Typescript
- Webpack 
- Redux
- Semantic UI
- React 
- React Hot Loader
- Express
- Sass
- MiniCssExtractPlugin
- Tslint

# Commands
Development
```
yarn dev
```

Production
```
yarn build && yarn start
```

Test
```
yarn test
```
